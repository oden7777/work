# Cloud Spanner

#### 概要
GCPで提供されているデータベース。
###### 歴史
2012年ごろからGoogle社内で利用されていた。
2017年7月には東京リージョンが提供されている。AdWordsやGooglePlayで利用されているらしい。

###### どんなサービス？
フルマネージドにスケール可能なRDBMSとして使えるサービス。
パフォーマンスを発揮するための設計上の制約があり。
RDBMSっぽく扱えるNoSQLと考えると色々しっくりくる。
